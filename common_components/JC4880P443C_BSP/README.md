# BSP: JC4880P443C_BSP

Project-local BSP for the JC4880P443C ESP32-P4 board.

This component merges the local board display/touch/audio BSP code and the
former `bsp_extra` audio helper layer into one component. The public
compatibility headers remain available:

- `bsp/esp-bsp.h`
- `bsp/display.h`
- `bsp/touch.h`
- `bsp_board_extra.h`

## Configuration

Configuration in `menuconfig`.

LCD panel is fixed to the ST7701 MIPI-DSI panel used by this project.

Selection color format `Board Support Package(JC4880P443C) --> Display --> Select LCD color format`
- RGB565 (default)
- RGB888
